# node-simpleChat

Simple chat with separate server and client based on socket.io

## Features 
1. Separate client/server
2. Wisper
3. Image sending

## Installation
1. `git clone`
2. `npm install` (Mongoose dev dependencies could be required like libkrb5-dev)
3. `bower install`
4. `node app.js`
5. Go `index.html` through your webserver
